# Smart Spenders – Personal Finance Management System

## Overview
Smart Spenders is a desktop-based personal finance management application that helps users track expenses, manage budgets, and monitor account balances efficiently.

## Features
- Expense and debit tracking
- Account balance management (Bank & Cash)
- Transaction recording system
- User-friendly interface for financial operations

## Technologies Used
- C#
- .NET (Windows Forms)
- SQL Server (Local Database)

## Purpose
The project is designed to assist users in making better financial decisions by organizing and analyzing their spending behavior.
